# 😂 Funny System Roast Page

A small joke website that looks like a system scan / hacker interface and displays a funny "roast result" about the user.

⚠️ This is purely for fun and memes. No real insults intended.

---

## 🌐 What it does

- Simulates a system scan
- Displays a fake analysis of the user
- Shows a humorous message like:
  - "UNFUNNIEST PERSON EVER 💀"
- Designed to look like a terminal / hacker screen
- Works on mobile browsers (optimized for phones)
